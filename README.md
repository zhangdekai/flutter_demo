# Object

仿制微信主页面的Flutter Demo.

This project is a Demo for learning Flutter.

内容主要包含 相关Flutter 技术的学习和演示.

Demo 中为了演练，穿插使用了2种状态管理lib, Provider and GetX， 也有单纯使用 StatefulWidget +
Flutter原有Navigator,
正式项目建议选择其中一种，统一管理应用。
